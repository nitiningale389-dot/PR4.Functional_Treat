# 📊 Functional Treat - Data Analyzer & Transformer

A Python-based console application that demonstrates the use of **functional programming concepts** while performing common data analysis operations.

This project allows users to input numerical data, analyze it, sort it, filter it, and calculate factorials using recursion through a simple menu-driven interface.

---

## 🚀 Features

- 📥 Input 1D numerical dataset
- 📋 Display dataset summary
- 🔢 Calculate factorial using recursion
- 🔍 Filter data using a threshold (List Comprehension)
- 📈 Sort data in ascending or descending order
- 📊 Display dataset statistics using Lambda Function
- 🔁 Interactive menu-driven program

---

## 🛠 Functional Programming Concepts Used

- **Functions**
- **Recursion**
- **Lambda Functions**
- **List Comprehension**
- **Built-in Functions**
  - `max()`
  - `min()`
  - `sum()`
  - `sorted()`
